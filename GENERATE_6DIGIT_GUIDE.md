# 🎲 คู่มือการสร้างเลข 6 หลัก

**ฟีเจอร์ใหม่:** Generate 6-Digit Numbers  
**เมื่อใช้:** หลังจากวิเคราะห์สำเร็จ เพื่อสร้างเลขเล่นหวย  
**เหมาะสำหรับ:** ผู้ที่ต้องการเลข 6 หลักเต็มเพื่อเล่นหวยรางวัลที่ 1

---

## 📊 เข้าใจก่อน: เลข 2D/3F/3B คืออะไร?

### **เลขหวยไทย:**
```
รางวัลที่ 1 = 6 หลัก
ตัวอย่าง: 1️⃣2️⃣3️⃣4️⃣5️⃣6️⃣

├─ 3F (เลขหน้า 3 ตัว) = 123
├─ 3B (เลขหลัง 3 ตัว) = 456
└─ 2D (เลขท้าย 2 ตัว) = 56
```

### **ถ้าคุณวิเคราะห์ 2D:**
```
ผลลัพธ์: 23 (เลขท้าย 2 ตัว)
├─ ต้องหา: เลข 4 หลักแรก (0000-9999)
└─ เลข 6 หลัก: 0000**23**, 0001**23**, ..., 9999**23**
```

### **ถ้าคุณวิเคราะห์ 3F:**
```
ผลลัพธ์: 123 (เลขหน้า 3 ตัว)
├─ ต้องหา: เลข 3 หลักหลัง (000-999)
└─ เลข 6 หลัก: **123**000, **123**001, ..., **123**999
```

### **ถ้าคุณวิเคราะห์ 3B:**
```
ผลลัพธ์: 456 (เลขหลัง 3 ตัว)
├─ ต้องหา: เลข 3 หลักแรก (000-999)
└─ เลข 6 หลัก: 000**456**, 001**456**, ..., 999**456**
```

---

## 🚀 วิธีใช้งาน: 3 ขั้นตอน

### **ขั้นตอนที่ 1: วิเคราะห์**
```
1. ไปหน้า "วิเคราะห์"
2. เลือก:
   ├─ ประเภทหวย (2D, 3F, หรือ 3B)
   ├─ History (จำนวนงวด)
   ├─ Reference Draw (วันออกรางวัลอ้างอิง)
   └─ Target Date (วันที่ต้องการทาย)
3. กด "วิเคราะห์ตอนนี้"
```

### **ขั้นตอนที่ 2: เลือกเลข**
```
หลังวิเคราะห์ดูผล → ไป Analysis Detail
ที่หน้านี้จะเห็น:

✅ ตัวเลือก A — ความถี่สูงสุด
   ├─ 23
   ├─ 45
   ├─ 67
   └─ ...

✅ ตัวเลือก B — คะแนนอันดับ
   ├─ 34
   ├─ 12
   ├─ 56
   └─ ...
```

### **ขั้นตอนที่ 3: สร้างเลข 6 หลัก**

#### **วิธี A: Hover ที่เลขแล้วคลิก**
```
1. Hover (นำเมาส์ไปชี้) ที่เลข เช่น "23"
2. จะเห็นปุ่ม "🎲 สร้าง 6 หลัก" ปรากฏ
3. คลิกปุ่มนั้น
4. แสดงเลข 6 หลักทั้งหมด (1000 เลข)
```

**ตัวอย่าง (2D):**
```
เลขที่เลือก: 23

ผลลัพธ์:
┌───────────────────────────────────┐
│ เลข 6 หลักทั้งหมด (1000)         │
├───────────────────────────────────┤
│ 000023  000123  000223  000323   │
│ 000423  000523  000623  000723   │
│ ...                               │
│ 999623  999723  999823  999923   │
└───────────────────────────────────┘
```

## 💡 ตัวอย่างการใช้งานจริง

### **ตัวอย่าง 1: วิเคราะห์ 2D (เลขท้าย 2 ตัว)**

```
ขั้นตอน:
1. เลือก "2D" → วิเคราะห์
2. ได้ผล Option A: 23, 45, 67, 89, 01
3. Hover ที่เลข "23" → คลิก "🎲 สร้าง 6 หลัก"
4. เห็นเลข 6 หลัก 1000 ตัว:
   └─ 000023, 000123, 000223, ..., 999923

ใช้ยังไง:
├─ เล่นหวย: เลือกบางตัวจาก 1000 เลขนี้
├─ เช่น: 123023, 456023, 789023
└─ สิ่งที่สำคัญ: ลงท้ายด้วย "23"
```

### **ตัวอย่าง 2: วิเคราะห์ 3F (เลขหน้า 3 ตัว)**

```
ขั้นตอน:
1. เลือก "3F" → วิเคราะห์
2. ได้ผล Option B: 123, 234, 345, 456, 567
3. Hover ที่เลข "123" → คลิก "🎲 สร้าง 6 หลัก"
4. เห็นเลข 6 หลัก 1000 ตัว:
   └─ 123000, 123001, 123002, ..., 123999

ใช้ยังไง:
├─ เล่นหวย: เลือกบางตัวจาก 1000 เลขนี้
├─ เช่น: 123456, 123789, 123012
└─ สิ่งที่สำคัญ: ขึ้นต้นด้วย "123"
```

### **ตัวอย่าง 3: วิเคราะห์ 3B (เลขหลัง 3 ตัว)**

```
ขั้นตอน:
1. เลือก "3B" → วิเคราะห์
2. ได้ผล Option A: 456, 567, 678, 789, 890
3. Hover ที่เลข "456" → คลิก "🎲 สร้าง 6 หลัก"
4. เห็นเลข 6 หลัก 1000 ตัว:
   └─ 000456, 001456, 002456, ..., 999456

ใช้ยังไง:
├─ เล่นหวย: เลือกบางตัวจาก 1000 เลขนี้
├─ เช่น: 123456, 456456, 789456
└─ สิ่งที่สำคัญ: ลงท้ายด้วย "456"
```
