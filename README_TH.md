## ⚡ 3 วิธีรัน

### **วิธี 1: GUI Launcher (ง่ายสุด!) ⭐**
```bash
python launcher.py
```
→ Window GUI ปรากฏ → คลิก "🚀 รัน Server"

---

### **วิธี 2: Batch/Shell Script (ดับเบิลคลิก)**

**Windows:**
```
ดับเบิลคลิก: run_windows.bat
```

**Mac/Linux:**
```
ดับเบิลคลิก: run_mac_linux.sh
```

---

### **วิธี 3: Command Line (Manual)**
```bash
# ไปโฟลเดอร์โปรเจค
cd /path/to/lottery-guide

# เปิด venv
source venv/bin/activate  # Mac/Linux
venv\Scripts\activate      # Windows

# รัน Server
python manage.py runserver

# เปิด Browser
http://localhost:8000
```
