## 🪟 Windows

### **วิธี 1: ดับเบิลคลิก (ง่ายสุด!) ⭐**

1. **เปิด File Explorer**
   - ไปที่ `lottery-guide/SCRIPTS/`

2. **ดับเบิลคลิก**
   - ไฟล์: `run_windows.bat`

3. **รอ 10-15 วินาที**
   - Command Prompt จะปรากฏและรันโปรเจค

4. **เปิด Browser**
   - ไปที่: **http://localhost:8000**
   - ต้องเห็น Dashboard ✓

5. **ปิด Server (ทีหลัง)**
   - กด Ctrl + C ในหน้า Command Prompt

---

### **วิธี 2: ใช้ GUI Launcher**

1. **เปิด Command Prompt**
   - Windows Key + R → พิมพ์ `cmd` → Enter

2. **ไปโฟลเดอร์โปรเจค**
   ```cmd
   cd C:\path\to\lottery-guide
   ```

3. **รันสคริป**
   ```cmd
   python SCRIPTS/launcher.py
   ```

4. **GUI Window ปรากฏ**
   - คลิก: "🚀 รัน Server"

5. **รอจนเห็น**
   - "✅ Server รันอยู่"

6. **เปิด Browser**
   - ไปที่: **http://localhost:8000**

7. **ปิด Server**
   - คลิก: "⏹️ ปิด Server" ในหน้า GUI

---

### **วิธี 3: Command Line Manual**

1. **เปิด Command Prompt**
   - Windows Key + R → พิมพ์ `cmd` → Enter

2. **ไปโฟลเดอร์โปรเจค**
   ```cmd
   cd C:\path\to\lottery-guide
   ```

3. **เปิด Virtual Environment**
   ```cmd
   venv\Scripts\activate
   ```

4. **รัน Server**
   ```cmd
   python manage.py runserver
   ```

5. **รอจนเห็น**
   ```
   Starting development server at http://127.0.0.1:8000/
   Quit the server with CTRL-BREAK
   ```

6. **เปิด Browser**
   - ไปที่: **http://localhost:8000**

7. **ปิด Server**
   - กด Ctrl + C ในหน้า Command Prompt

## ⚠️ ปัญหาทั่วไป

### **❌ "Port 8000 already in use"**
- Port 8000 ใช้งานแล้ว รันที่พอร์ตอื่น:
```bash
python manage.py runserver 8080
# เข้าที่ http://localhost:8080
```

### **❌ "Connection refused"**
- Server ไม่รัน ลองรันสคริปใหม่

### **❌ "Module not found errors"**
- Dependencies ไม่ติดตั้ง รันติดตั้งใหม่:
```bash
source venv/bin/activate  # หรือ venv\Scripts\activate
pip install -r requirements.txt
```

### **❌ "Database locked"**
- ลบ `db.sqlite3` และรันไฟล์นี้ใหม่

## 💡 Tips

- **Server รันเพื่อ Development เท่านั้น** (ไม่ใช่ Production)
- **ปิด Server ใดขณะไม่ใช้** เพื่อประหยัดทรัพยากร
- **รันได้หลายครั้ง** ไม่มีปัญหา
- **ไม่ต้องติดตั้งใหม่** หลังครั้งแรก (บันทึกไป DOCS/INSTALLATION_GUIDE_TH.md)
