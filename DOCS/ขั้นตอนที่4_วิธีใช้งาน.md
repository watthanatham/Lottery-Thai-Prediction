# 📊 ขั้นตอนที่ 4: วิธีใช้งาน LottoAI

**เป้าหมาย:** เรียนรู้วิธีวิเคราะห์หวยและสร้างเลขเล่น

**สถานะ:** Server รันอยู่ ✓

---

## 🎯 ขั้นตอนเร็วสุด (5 นาที)

### **1. ไปที่ Dashboard**
- ที่ **http://localhost:8000**

### **2. ป้อนข้อมูล Lottery Draw**
- เลือก: **ประเภทหวย** (2D, 3F, 3B, 6D)
- ป้อน: **เลขที่ออก** (ผลรางวัล)
- ป้อน: **วันที่** (ไม่บังคับ)
- คลิก: **บันทึก**

### **3. ไปวิเคราะห์**
- คลิก: **"วิเคราะห์"** (ปุ่มสีเขียว)

### **4. เลือกตัวเลือก A หรือ B**
- **Option A:** ตัวเลขที่ออกเยอะที่สุดจากสูตร 75 ตัว
- **Option B:** ตัวเลขที่ได้คะแนนสูงสุดจากการประเมิน
- ดูรายละเอียด → คลิก: **"🎲 สร้าง 6 หลัก"**

### **5. สร้างเลข 6 หลัก**
- ได้ 1000 ชุดเลขเล่น
- ปุ่ม:
  - **📋 Copy All** → คัดลอกทั้งหมด
  - **⬇️ Download** → ดาวน์โหลด CSV
  - **🖨️ Print** → พิมพ์

---

## 📚 คู่มายาว (30 นาที)

### **Part 1: บันทึกผลรางวัล**

**ที่ Dashboard:**
1. Scroll down หา "📥 บันทึกผลออกรางวัล"
2. กรอก:
   - ✍️ **ประเภทหวย:** เลือก (2D/3F/3B/6D)
   - ✍️ **เลขที่ออก:** เช่น `49` (2D), `123` (3F), `1234` (3B)
   - ✍️ **วันที่:** ไม่บังคับ (หรือ Select date ถ้าอยากบันทึก)
3. คลิก: **บันทึก**
4. ข้อมูลปรากฏใน "ประวัติการออกรางวัล" ด้านบน

---

### **Part 2: วิเคราะห์**

**ขั้นตอน:**
1. คลิก: **"วิเคราะห์"** (ปุ่มสีเขียว)
2. ปรากฏหน้า "Analysis" พร้อม Form
3. **เลือกสูตร (Optional):**
   - ปกติปิด "เลือกสูตรที่ต้องการ"
   - ถ้าเปิด → select เฉพาะสูตรที่ต้อง
4. **ป้อน Target Draw Date (Optional):**
   - วันที่คุณอยากวิเคราะห์ (ปกติปิด ใช้เลขล่าสุด)
5. **เลือก Lottery Type:**
   - 2D / 3F / 3B / 6D
6. คลิก: **"🔍 วิเคราะห์"**

---

### **Part 3: ผลลัพธ์ - Option A vs B**

**หน้า Analysis Detail แสดง:**

#### **Option A: ความถี่ (Frequency)**
- นับจำนวนครั้ง ตัวเลขปรากฏใน 75 สูตร
- **ตัวเลขที่นับได้เยอะ** = น่าจะออก

```
ตัวอย่าง:
ตัวเลข 1 → ออกใน 45 สูตร (เยอะที่สุด)
ตัวเลข 2 → ออกใน 40 สูตร
ตัวเลข 3 → ออกใน 30 สูตร
```

**วิธีใช้:**
- ดูท็อป 3 ตัวเลขที่นับได้เยอะ
- นำมาสร้างเลข 6 หลัก (ดูส่วน "สร้าง 6 หลัก")
- เล่นผลรวม: โอกาสสูง แต่รสชาติอ่อน

#### **Option B: คะแนน (Score)**
- ให้คะแนนตัวเลขตามตำแหน่ง:
  - 🥇 1st (ออกใน 1 สูตร): **5 คะแนน**
  - 🥈 2nd (ออกใน 2 สูตร): **4 คะแนน**
  - 🥉 3rd (ออกใน 3 สูตร): **3 คะแนน**
  - ออกใน 4-5 สูตร: **2 คะแนน**
  - ออกใน 6+ สูตร: **1 คะแนน**

**วิธีใช้:**
- ดูท็อป 3 ตัวเลขที่ได้คะแนนสูง
- น้ำหนัก: ระหว่าง A + B

---

### **Part 4: สร้างเลข 6 หลัก**

**ที่หน้า Analysis Detail:**

1. **Scroll ลงไป** ไปหา Option A หรือ B
2. Hover ที่บล็อก → ปรากฏ: **"🎲 สร้าง 6 หลัก"**
3. คลิก → ไปหน้า "Generate 6 Digits"

**หน้า Generate:**
- แสดง **1000 ชุด** ตัวเลขเล่น
- ตัวอย่าง (2D):
  ```
  1249, 1349, 1449, 1549, ... (1000 ชุด)
  ```

**ปุ่ม:**
- 📋 **Copy All** → Ctrl+C อัตโนมัติ
- ⬇️ **Download** → CSV file (ใช้ Excel ได้)
- 🖨️ **Print** → พิมพ์กระดาษ

---

### **Part 5: ตรวจสอบผล**

**ทีหลังหลังออกรางวัล:**

1. ไปหน้า **"ตรวจสอบ"** (Verification)
2. ป้อน:
   - **ผลรางวัลจริง** ที่ออก
   - **เลขที่คุณเล่น** จากการวิเคราะห์
3. System บอก: **ถูก / ผิด / ถูกบาง**

---

## 🎲 3 วิธีเลือกตัวเลข

### **วิธี 1: ใช้ Option A อย่างเดียว**
- ✅ **ได้เปรียบ:** ตรง ตรง ตรง 100% จากสูตร
- ❌ **ด้อย:** อาจ Conservative (หลีกเลี่ยง Risk)
- **ใช้เมื่อ:** ต้องการความแน่นอนสูง

### **วิธี 2: ใช้ Option B อย่างเดียว**
- ✅ **ได้เปรียบ:** ดูเลขที่ "พิเศษ" ที่ออกน้อยแต่คุณ
- ❌ **ด้อย:** ก็อาจลิ่มตัวจากโหลกโจทย์
- **ใช้เมื่อ:** ชอบ Aggressive (High Risk High Reward)

### **วิธี 3: ผสม A + B (แนะนำ!)**
- ใช้ทั้งสองสูตร → ดูเลขที่ปรากฏในทั้ง A และ B
- ✅ **ได้เปรียบ:** สมดุล / ปลอดภัย / ผ่อง
- **วิธี:** สร้าง 6 หลักจาก A → สร้าง 6 หลักจาก B → หา "Cross-over" (เลขปรากฏใน 2 ที่)

---

## 📊 ทำความเข้าใจผลลัพธ์

### **สูตร (Formula) คืออะไร?**
- 75 สูตรคณิตศาสตร์ที่ช่วยทำนายแนวโน้ม
- แต่ละสูตร predict ตัวเลขต่างกัน
- รวมกัน → ได้ "ภาพรวม" ที่ถูกต้องสูง

### **Confidence Level คืออะไร?**
- ประกาศความเชื่อมั่น ของผลวิเคราะห์
- 🟢 ✓ High (80-100%) = มั่นใจสูง
- 🟡 ⚠️ Medium (50-80%) = ปานกลาง
- 🔴 ❌ Low (<50%) = ระวัง / อาจต้องหลีกเลี่ยง

---

## 💡 เคล็ดลับ

### **🎯 ใช้งานได้ดี**
1. **บันทึก Lottery Draw บ่อยๆ** → System มีข้อมูลมากขึ้น → ผลดีขึ้น
2. **ดูประวัติการวิเคราะห์** → เห็น Accuracy เป็นไง
3. **ผสม A + B** → ผลมีสมดุล
4. **ไม่เล่นทั้งหมด 1000 ชุด** → เลือกเฉพาะ A หรือ B ท็อป 10-20 ชุด
5. **Follow Confidence Level** → ถ้า Low ให้หลีกเลี่ยง

### **⚠️ ขึ้นใจจริง**
- 🚫 **ไม่รับประกัน 100%** = หวยคือเกม
- 🚫 **ไม่ต้องเล่นใหญ่เพราะ AI** = เล่นตามสติ
- 🚫 **ดูข้อมูลเก่า** = ไม่ validate ผลลัพธ์
- ✅ **ใช้เป็นชี้แนะ** เท่านั้น
