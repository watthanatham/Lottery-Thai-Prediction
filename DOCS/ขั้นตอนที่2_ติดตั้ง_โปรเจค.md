# 📦 ขั้นตอนที่ 2: ติดตั้งโปรเจค

**เป้าหมาย:** ติดตั้ง LottoAI และ Dependencies ของมัน

**สถานะ:** Python 3.10+ ติดตั้งแล้ว ✓

---

## 🪟 Windows

### **ขั้นตอนแบบง่าย: ใช้ Script (แนะนำ!)**

1. **เปิด File Explorer**
   - ไปที่โฟลเดอร์ `lottery-guide` ของคุณ

2. **ดับเบิลคลิก Script**
   - หา: `SCRIPTS/run_windows.bat`
   - ดับเบิลคลิก

3. **รอ 30-60 วินาที**
   - Script จะทำ:
     - ✓ สร้าง Virtual Environment (venv)
     - ✓ ติดตั้ง Dependencies
     - ✓ สร้างฐานข้อมูล
     - ✓ สร้าง Admin Account (ให้คุณกรอก)
   
4. **กรอก Admin Username และ Password**
   - ติดตั้งจะให้คุณกรอก Username และ Password
   - จำไว้ใช้เข้า Admin Page

5. **ตรวจสอบการติดตั้ง**
   - Command Prompt ปิดเอง = สำเร็จ ✓
   - เห็นข้อความ "✅ ติดตั้งแล้ว" = สำเร็จ ✓

---

### **ขั้นตอนแบบ Manual (ถ้า Script ไม่ได้)**

1. **เปิด Command Prompt**
   - Windows Key + R → พิมพ์ `cmd` → Enter
   - หรือ Shift + Right-click ในโฟลเดอร์ → "Open PowerShell here"

2. **ไปโฟลเดอร์โปรเจค**
   ```cmd
   cd C:\path\to\lottery-guide
   ```

3. **สร้าง Virtual Environment**
   ```cmd
   python -m venv venv
   ```

4. **เปิด Virtual Environment**
   ```cmd
   venv\Scripts\activate
   ```
   - ต้องเห็น `(venv)` ที่หน้า Command Prompt

5. **ติดตั้ง Dependencies**
   ```cmd
   pip install -r requirements.txt
   ```

6. **สร้างฐานข้อมูล**
   ```cmd
   python manage.py migrate
   ```

7. **สร้าง Admin Account**
   ```cmd
   python manage.py createsuperuser
   ```
   - กรอก Username ของคุณ
   - กรอก Email
   - กรอก Password 2 ครั้ง

**✅ เสร็จ!** ไปขั้นตอนที่ 3

## ❓ ปัญหาใจ

### **❌ "pip: command not found"**
- ต้องเปิด Virtual Environment ก่อน: `source venv/bin/activate` (Mac/Linux) หรือ `venv\Scripts\activate` (Windows)

### **❌ "Module not found"**
- ลบโฟลเดอร์ venv ทั้งหมด
- รันติดตั้งใหม่ (วิธี Script แนะนำ)

### **❌ "Permission denied" (Mac/Linux)**
- รัน: `chmod +x SCRIPTS/run_mac_linux.sh`

### **❌ Database error**
- ลบไฟล์ `db.sqlite3` ถ้ามี
- รันติดตั้งใหม่
