## 💡 วิธีใช้ Documentation

### **ขั้นตอน = Step-by-Step**
- อ่านเรียงลำดับ
- รวม Screenshots / Code
- ตอบสำหรับสถานการณ์ต่างๆ (Windows/Mac/Linux)

### **ANALYSIS_GUIDE = รายละเอียด**
- อธิบาย Option A และ B อย่างลึก
- ตัวอย่างการใช้งาน
- เคล็ดลับขั้นสูง

### **QUICK_REFERENCE = Cheat Sheet**
- ตารางเปรียบเทียบ
- คำสั่งด่วน
- พิมพ์ได้ / ใช้ Mobile

### **FAQ = ตอบคำถาม**
- Q&A ทั่วไป
- ปัญหาและแก้
- Edge Cases
