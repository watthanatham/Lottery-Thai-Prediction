# 📖 คู่มือการใช้งาน Analysis Detail Page

## 📌 บทนำ
หน้า **Analysis Results** (ผลวิเคราะห์) แสดงการคำนวณจาก **75 สูตรหวย** เพื่อช่วยให้คุณหาแนวทางเลขที่น่าจะออก

---

## 🎯 ผลวิเคราะห์ 2 ตัวเลือก

### **ตัวเลือก A: ความถี่สูงสุด (Highest Frequency)**

#### 🔍 หมายความว่า:
- **นับจำนวน** ว่าแต่ละเลขปรากฏในอันดับ Top-5 ของสูตรกี่ครั้ง
- เลขที่ถูกสูตรหลายๆ ตัวเลือก = เลขที่หลายคนรู้สึกว่าน่าจะออก
- **ยิ่งมี "formulas" มาก = น่าเชื่อถือมากขึ้น**

#### 📊 วิธีอ่าน:
```
เลข 23
├─ 12 formulas    ← สูตรนี้คำนวณได้เลข 23 ถึง 12 สูตร จาก 75 สูตร
├─ Rank #1        ← อันดับที่ 1 (ดีที่สุด)
└─ ✓ Hit          ← ตอนเดิมออก 23 จริง ✓
```

#### 💡 ใช้ยังไง:
- **ดีที่สุด:** เลขที่มี **12+ formulas** → ลองเล่น
- **ปานกลาง:** เลขที่มี **7-11 formulas** → พิจารณา
- **ความเสี่ยง:** เลขที่มี **น้อยกว่า 7 formulas** → อาจจะเสี่ยง

---

### **ตัวเลือก B: คะแนนอันดับ (Ranking Score)**

#### 🔍 หมายความว่า:
- **ให้น้ำหนัก** แต่ละเลขตามลำดับที่ออก
- เลขอันดับที่ 1 = **5 คะแนน**
- เลขอันดับที่ 2 = **4 คะแนน**
- เลขอันดับที่ 3 = **3 คะแนน**
- เลขอันดับที่ 4 = **2 คะแนน**
- เลขอันดับที่ 5 = **1 คะแนน**

**รวมคะแนนสูงสุด = 5 × 75 = 375 คะแนน**

#### 📊 วิธีอ่าน:
```
เลข 45
├─ 320 pts        ← รวมคะแนน 320 คะแนน (สูงมาก!)
├─ Rank #1        ← อันดับที่ 1
└─ ✓ Hit          ← เคยออก ✓
```

#### 💡 ใช้ยังไง:
- **ดีที่สุด:** **250+ คะแนน** → เลขแรงที่สุด
- **ปานกลาง:** **150-249 คะแนน** → พอสมควร
- **อ่อน:** **น้อยกว่า 150 คะแนน** → อาจลอย

---

## 🎲 วิธีเลือกเลขเล่น

### **วิธีที่ 1: ใช้ Option A โดยตัวเอง**
```
ดูจากเลขที่มี "formulas" มากที่สุด
└─ เลือก 3-5 เลขแรก เล่นแบบ "มั่นใจสูง"
```

### **วิธีที่ 2: ใช้ Option B โดยตัวเอง**
```
ดูจากเลขที่มี "pts" สูงสุด
└─ เลือก 3-5 เลขแรก เล่นแบบ "คุณภาพสูง"
```

### **วิธีที่ 3: รวม A + B (ชาญฉลาด) ⭐ ★ ★ ★ ★**
```
🎯 ค้นหาเลขที่ปรากฏใน Top-5 ของทั้ง A และ B
   └─ เลขเหล่านี้ได้รับการยอมรับจากหลายๆ มุมมอง
   └─ น้ำหนักสูง + ความถี่สูง = ความมั่นใจสูง ✓

ตัวอย่าง:
├─ Option A: 23, 45, 67, 89, 01
├─ Option B: 23, 34, 45, 56, 78
└─ 🌟 ตัวเลขที่ซ้ำ (เหมือนกัน): 23, 45
   → แนะนำเล่น 23 และ 45 ก่อน!
```

## 📋 Formula Breakdown คืออะไร?

### 🔧 "ตารางแบ่งแยกสูตร"
- แสดงผลลัพธ์ Top-5 ของแต่ละสูตรทั้ง 75 ตัว
- ช่วยให้คุณเห็นว่า **สูตรไหนออกหลังจากสูตรไหน**

### 📊 วิธีอ่าน:
```
Code: F001
Formula: ผลรวมตัวเลขหลัก
Group: คำนวณ
Top-5 Predictions: 23, 45, 67, 12, 34
```

### 💡 วิธีใช้:
```
ถ้าคุณสงสัยว่า "เลข 23 มาจากสูตรไหน"
└─ ดูตารางแต่ละแถว
└─ หาเลข 23 ใน "Top-5 Predictions"
└─ ดูชื่อสูตรที่สนับสนุนเลขนั้น
```

## ✅ ตรวจสอบผลกับวันการออกรางวัลจริง

### ขั้นตอน:
1. **ทำการวิเคราะห์** เลขที่คาดว่าจะออก
2. **รอการออกรางวัล** วันถัดไป
3. ไปที่หน้า **"ตรวจสอบ" (Checklist)**
4. เลือก **Analysis Session** ที่ต้องการตรวจสอบ
5. เลือก **วันการออกรางวัลจริง**
6. กด **ตรวจสอบ** → ระบบจะบอกว่าเดาถูกเลขไหน

### 📊 ผลลัพธ์:
- ✓ **Hit** = เดาถูก (เลขออกจริง)
- ✗ **Miss** = เดาผิด (เลขไม่ออก)

## 📈 เคล็ดลับการใช้งาน

### 1️⃣ **เลือกจำนวน History ที่เหมาะสม**
```
少 (5-10 draws)   → วิเคราะห์เร็ว, แต่อาจไม่แม่นยำ
중 (20-50 draws)  → สมดุล ⭐ (แนะนำ)
多 (100+ draws)   → แม่นยำสูง, แต่เวลาวิเคราะห์นาน
```

### 2️⃣ **อย่าเชื่อสูตร 100%**
```
สูตรเป็นแค่ "แนวทาง" ไม่ใช่ "รับประกัน"
├─ ลองเล่นเพียง 2-3 เลขจากแต่ละตัวเลือก
└─ ไม่ควรเล่นมากกว่า 10 เลขต่อครั้ง
```

### 3️⃣ **เปรียบเทียบ A กับ B ช่วยตัดสินใจ**
```
ถ้า Option A และ B เห็นด้วย → น้ำหนักมาก ⭐
ถ้า Option A เท่านั้น          → พิจารณา
ถ้า Option B เท่านั้น          → พิจารณา
```

### 4️⃣ **ติดตามประวัติ**
```
ไปที่ "ประวัติการวิเคราะห์" (Analysis History)
ดูว่าสูตรของคุณแม่นยำเพียงไร
└─ ร้อยละถูก = ความเชื่อมั่น
```

## ❌ อาจเกิดอะไรขึ้น?

### ❓ "เลขเหมือนกันในทั้ง A และ B"
```
ดี! → เลขนั้นมีน้ำหนักสูงจริง ✓
```

### ❓ "Option A เต็มไป เลขหลาย ๆ ตัว"
```
อาจเพราะว่า:
├─ สูตรต่างใจเห็นเลขเดียวกัน
└─ ลองใช้ Option B ซึ่งเล็งแม่นกว่า
```

### ❓ "ไม่มีเลขไหนสูงที่สุด"
```
อาจเป็นเพราะ:
├─ ข้อมูล History ไม่เพียงพอ (ลองใช้ 50+ draws)
├─ ลองเปลี่ยน Reference Draw
└─ ลองใช้สูตรประเภทอื่น (2D, 3F, 3B)
```

## 🎓 สรุป (TL;DR)

| แนวทาง | วิธี | ความเสี่ยง |
